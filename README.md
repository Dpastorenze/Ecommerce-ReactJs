### INSTALACIÒN
# Pasos a seguir
- Clonar repositorio 
    - git clone https://github.com/Dpastorenze/Ecommerce-ReactJs.git
- Moverse a la carpeta 
    - cd Ecommerce-ReactJs
- Instalar las dependencias
    - npm install o npm init o npm i vite 
- Compilar
    - npm run dev
